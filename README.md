## What is Money Protocol?

Money Protocol is an autonomous DeFi system built on Solana. Three programs. Two wallets. One system that manages capital, rebalances positions, and distributes yield — without human intervention.

Idle capital is put to work. Holders cheer it. The agent runs the fund.

## Three Programs. Two Wallets.

| Program | Description |
|---|---|
| `FundVault` | Holds capital. Issues LP shares. Immutable. |
| `AgentRouter` | Routes agent instructions. Enforces strategy bounds. |
| `YieldDistributor` | Harvests, splits, and distributes yield each epoch. |
