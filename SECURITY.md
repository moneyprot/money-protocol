# Security Policy

## Reporting a Vulnerability

Money Protocol takes security seriously. If you discover a vulnerability, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

### How to Report

Contact us privately via:
- Twitter DM: [@MoneyProt](https://x.com/MoneyProt)
- Telegram: [t.me/moneyprot](https://t.me/moneyprot)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact assessment
- Your suggested fix (if any)

We will respond within 48 hours and work with you on a fix before public disclosure.

## Bug Bounty Program

Money Protocol will launch a public bug bounty program prior to mainnet deployment.

| Severity | Description | Reward |
|---|---|---|
| Critical | Funds at direct risk of loss | Up to $100,000 |
| High | Protocol functionality severely impaired | Up to $25,000 |
| Medium | Limited impact on protocol operations | Up to $5,000 |
| Low | Minor issues or informational findings | Up to $500 |
